# üçï Pizzeria ‚Äì MEAN Stack Application

A full-stack Pizzeria web application built using the **MEAN stack**  
(**MongoDB, Express, Angular, Node.js**).

This project demonstrates end-to-end application development including REST APIs, database persistence, and a modern Angular frontend.

---

## üìå Overview

The Pizzeria application allows users to:
- Browse available pizzas (Veg / Non-Veg)
- Build custom pizzas using ingredients
- Manage cart items with quantity control
- Persist cart data using MongoDB
- View total pricing dynamically

The focus of this project is **functionality, architecture, and data flow**, with UI styling kept minimal for clarity.
